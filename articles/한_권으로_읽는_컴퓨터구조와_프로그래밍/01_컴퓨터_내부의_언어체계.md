# 컴퓨터 내부의 언어 체계: 컴퓨터는 어떤 말을 사용할까

프로그래머의 할 일은 컴퓨터에게 명령을 내리는 것이다

## 언어란 무엇인가

편의를 제공하기 위한 지름길

## 문자 언어

문자 언어는 기호를 나열한 것이다

```
'맛집' -> ㅁㅏㅅㅈㅣㅂ, 'yum' -> y u m
```

세 가지 구성요소가 문자 언어의 틀을 이룬다 (컴퓨터 언어도 마찬가지)

- 기호가 들어갈 상자
- 상자에 들어갈 기호
- 상자의 순서

## 비트

### 상자

자연어에서는 이 상자를 문자(character)라고 부른다

컴퓨터 언어에서는 이 상자를 비트(bit)라고 부른다

### 비트
```
비트(bit) = 2진법(binary) + 숫자(digit)
```

비트를 사용하면 적은 비용으로 편리하게 기호를 담을 수 있다

*컴퓨팅: 컴퓨터를 사용한 계산

## 논리 연산

비트 사용법 중 하나는 예/아니오 로 답변될 수 있는 질문에 대한 답을 표현하는 것이다

```
예 -> true
아니오 -> false
```

다른 비트들이 표현하는 내용으로부터 새로운 비트를 만들어내는 동작

### 불리언 대수

*대수: 수에 대한 연산 규칙의 집합

불리언 대수(Boolean algebra) By George Boole

비트에 대헤 사용할 수 있는 연산 규칙의 집합

#### 기본적인 불리언 연산자

- NOT: 논리적 반대 true -> false, false -> true
- AND: 첫번째와 두번째 비트가 모두 참일 경우에만 결과가 참이 된다
- OR: 첫번째 비트와 두번째 비트 둘 중 하나가 참일 경우에 결과가 참이 된다
- XOR: 첫번째 비트와 두번째 비트가 다를 경우 참이 된다 (exclusive OR)

### 드모르간의 법칙

Augustus De Morgan이 추가로 알아낸 불리언 대수에 적용할 수 있는 법칙

```
a AND b === NOT(NOT a or NOT b)
```

NOT을 충분히 잘 사용하면 AND 연산을 OR 연산으로 대신할 수 있고 역으로 OR을 AND로 대신할 수 있다

컴퓨터에서 입력을 항상 원하는 형태로 얻을 수만은 없기 때문에 유용할 수 있다
